# Ngôn ngữ lập trình Rust

Tài liệu học ngôn ngữ lập trình Rust bằng tiếng Việt. Nội dung được xây dựng với nguyên tắc học hiểu
bản chất, nội dung rõ ràng có cấu trúc, tìm hiểu sâu nhưng không lan man.

## Giới thiệu

Rust là một ngôn ngữ lập trình hiện đại và an toàn, viết code giống như một ngôn ngữ lập trình bậc
cao nhưng đem lại hiệu năng của một ngôn ngữ lập trình bậc thấp (C-level).

Sách được viết nhằm mục đích học tập và ghi chú cá nhân, nên không tránh khỏi sai sót. Mình hi vọng
tài liệu sẽ giúp được mọi người học tập tốt hơn, dễ tiếp cập hơn và cũng mong những đóng góp cải
thiện từ mọi người để cuốn sách ngày càng hoàn thiện.
